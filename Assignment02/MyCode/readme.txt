Follow the steps below so that you can build and run my program.

1. Compile:
    In this folder, type the following command: $make
    After that, there will be exe file named "Floorplan" in this folder.
    
2. Run Program:
    In this folder, type the following command: $./Floorplan [input file] [output file]
    e.g. $./Floorplan sample.in sample.out
    The program would print some information on console so that user see which part have been executed so far.
