Follow the steps below so that you can build and run my program.

1. Compile:
    In this folder, type the following command: $make
    After that, there will be exe file named "Lab4" in this folder.
    
2. Run Program:
    In this folder, type the following command: $./Lab4 [input file] [output file]
    e.g. $./Lab3 sample.in sample.out
